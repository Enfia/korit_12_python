### 파이썬 자료형 
1. 리스트[] : 값의 추가·수정·삭제가 많음 / 순서가 중요한 경우 사용
2. 튜플() : 변경되면 안 되는 고정 데이터들을 순서대로 관리할 때 사용 / 추가 수정 삭제 불가능
3. 세트 set({}) : 중복을 제거하거나 포함 여부를 빠르게 확인할 때 사용
4. 딕셔너리{} : 키를 기준으로 값을 빠르게 조회·관리해야 할 때 사용

### list 요소 추가와 삭제
- append() - 항상 마지막 인덱스에 요소를 추가
- insert(인덱스, 값) - 해당 인덱스에 요소를 추가
- pop() - 매개변수가 없으면 마지막의 요소가 삭제
- pop(인덱스) - 해당 인덱스의 요소를 삭제
- remove(값) - 해당 값을 찾아서 삭제
