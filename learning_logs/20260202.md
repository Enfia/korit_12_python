### 파이썬 자료형 
1. 리스트[] : 값의 추가·수정·삭제가 많음 / 순서가 중요한 경우 사용
2. 튜플() : 변경되면 안 되는 고정 데이터들을 순서대로 관리할 때 사용 / 추가 수정 삭제 불가능
3. 세트 set({}) : 중복을 제거하거나 포함 여부를 빠르게 확인할 때 사용
4. 딕셔너리{} : 키를 기준으로 값을 빠르게 조회·관리해야 할 때 사용

### 파이썬 시퀀스 / 비시퀀스
1. 시퀀스는 리스트, 튜플 (인덱스가 있음)
2. 비시퀀스는 세트, 딕셔너리 (인덱스가 없음)

### list 요소 추가와 삭제
- append(값) - 항상 마지막 인덱스에 요소를 추가
- insert(인덱스, 값) - 해당 인덱스에 요소를 추가
- pop() - 매개변수가 없으면 마지막의 요소가 삭제
- pop(인덱스) - 해당 인덱스의 요소를 삭제
- remove(값) - 해당 값을 찾아서 삭제

### 튜플
- title() - 이어줌

### 세트
- add(값) - 값 추가
- remove(값) - 해당 값 삭제 없는 값을 받으면 오류 발생
- discard(값) - 없는 값을 넣어도 오류가 발생하지 않음

### 딕셔너리
- keys() - 키들을 보여줌(list로 형변환 필요)
- values() - 값들을 보여줌(list로 형변환 필요)
- pop(키) - 삭제한 값을 보여줌
